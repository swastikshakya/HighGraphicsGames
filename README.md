# 🎮 High Graphics 3D Game

A browser-based 3D game built with **Three.js**, featuring real-time rendering, lighting, shadows, and a large 3D environment.

## ✨ Features

* 🌍 Real-time 3D graphics
* 💡 Dynamic lighting
* 🌫️ Fog effects
* 🌳 Large terrain with trees
* 🎥 Perspective camera
* ⚡ Hardware-accelerated WebGL rendering
* 📱 Responsive design
* 🎮 Basic player movement (WASD)
* 🚀 Easy to extend with new gameplay features

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js
* WebGL

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/HighGraphicsGame.git
```

### 2. Open the Project

```bash
cd HighGraphicsGame
```

### 3. Run a Local Server

Using VS Code Live Server:

* Install the **Live Server** extension.
* Right-click on `index.html`.
* Select **Open with Live Server**.

Or using Node.js:

```bash
npx serve
```

Then open the URL shown in your terminal.
